Source Plugin Demos

There are 4 html files in this demo:
  • demo.html (Test scaffold that loads demo_plugin.html)
    • Load the demo with delayed 'init()'
  • backend_demo.html(Test scaffold that loads backend_demo_plugin.html
  • demo_plugin.html (Demo Source plugin that performs the SourceBridge handshake and displays the data received from the parent window).
  • backend_demo_plugin.html (Demo Source plugin that communicates with a demo backend)
